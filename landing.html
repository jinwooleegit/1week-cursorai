랜딩 페이지 예제

프로젝트 개요

이 예제는 Cursor AI를 활용하여 만든 랜딩 페이지입니다. 제품이나 서비스를 효과적으로 소개하고 전환율을 높이는 데 중점을 두었습니다.

주요 기능

  • 강력한 히어로 섹션
  • 제품/서비스 특징 소개
  • 고객 후기 섹션
  • 행동 유도(CTA) 버튼

코드 예시

HTML 구조

<section class="hero">
    <div class="container">
        <div class="hero-content">
            <h1>제품/서비스 이름</h1>
            <p class="hero-subtitle">강력한 제품/서비스 설명</p>
            <div class="cta-buttons">
                <a href="#contact" class="btn btn-primary">시작하기</a>
                <a href="#features" class="btn btn-secondary">자세히 보기</a>
            </div>
        </div>
        <div class="hero-image">
            <img src="hero-image.png" alt="제품/서비스 이미지">
        </div>
    </div>
</section>

JavaScript 기능

// 스크롤 애니메이션
const animateOnScroll = () => {
    const elements = document.querySelectorAll('.animate-on-scroll');
    
    elements.forEach(element => {
        const elementPosition = element.getBoundingClientRect().top;
        const screenPosition = window.innerHeight;
        
        if (elementPosition < screenPosition) {
            element.classList.add('animate');
        }
    });
};

window.addEventListener('scroll', animateOnScroll);

실행 방법

  1. Cursor IDE에서 새 프로젝트 생성
  2. AI에게 "히어로 섹션이 있는 현대적인 랜딩 페이지 만들기" 명령어 입력
  3. 생성된 코드 검토 및 수정
  4. 이미지 및 콘텐츠 추가
  5. CTA 버튼 및 폼 최적화
