포트폴리오 웹사이트 예제

프로젝트 개요

이 예제는 Cursor AI를 활용하여 만든 포트폴리오 웹사이트입니다. 반응형 디자인과 모던한 UI/UX를 구현했습니다.

주요 기능

  • 반응형 그리드 레이아웃
  • 프로젝트 필터링 시스템
  • 이미지 갤러리
  • 연락처 폼

코드 예시

HTML 구조

<section class="portfolio">
    <div class="portfolio-filters">
        <button class="filter-btn active" data-filter="all">전체</button>
        <button class="filter-btn" data-filter="web">웹</button>
        <button class="filter-btn" data-filter="app">앱</button>
        <button class="filter-btn" data-filter="design">디자인</button>
    </div>
    <div class="portfolio-grid">
        <div class="portfolio-item" data-category="web">
            <img src="project1.jpg" alt="프로젝트 1">
            <div class="portfolio-overlay">
                <h3>프로젝트 제목</h3>
                <p>프로젝트 설명</p>
            </div>
        </div>
        // ... 더 많은 프로젝트 아이템
    </div>
</section>

CSS 스타일

.portfolio-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 2rem;
    padding: 2rem 0;
}

.portfolio-item {
    position: relative;
    overflow: hidden;
    border-radius: var(--border-radius);
    box-shadow: var(--shadow);
}

.portfolio-overlay {
    position: absolute;
    bottom: 0;
    left: 0;
    right: 0;
    background: rgba(0, 0, 0, 0.8);
    padding: 1.5rem;
    transform: translateY(100%);
    transition: transform 0.3s ease;
}

.portfolio-item:hover .portfolio-overlay {
    transform: translateY(0);
}

JavaScript 기능

// 필터링 기능
const filterButtons = document.querySelectorAll('.filter-btn');
const portfolioItems = document.querySelectorAll('.portfolio-item');

filterButtons.forEach(button => {
    button.addEventListener('click', () => {
        const filter = button.dataset.filter;
        
        filterButtons.forEach(btn => btn.classList.remove('active'));
        button.classList.add('active');
        
        portfolioItems.forEach(item => {
            if (filter === 'all' || item.dataset.category === filter) {
                item.style.display = 'block';
            } else {
                item.style.display = 'none';
            }
        });
    });
});
