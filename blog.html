블로그 웹사이트 예제

프로젝트 개요

이 예제는 Cursor AI를 활용하여 만든 블로그 웹사이트입니다. 콘텐츠 관리와 사용자 경험에 중점을 두었습니다.

주요 기능

  • 반응형 그리드 레이아웃
  • 카테고리 기반 필터링
  • 검색 기능
  • 댓글 시스템

코드 예시

HTML 구조

<div class="blog-grid">
    <article class="blog-card">
        <div class="blog-image">
            <img src="blog-image.jpg" alt="블로그 이미지">
        </div>
        <div class="blog-content">
            <div class="blog-meta">
                <span class="category">개발</span>
                <span class="date">2023-12-01</span>
            </div>
            <h3>블로그 제목</h3>
            <p>블로그 내용 요약...</p>
            <a href="#" class="read-more">더 읽기</a>
        </div>
    </article>
</div>

JavaScript 기능

// 카테고리 필터링
const filterPosts = (category) => {
    const posts = document.querySelectorAll('.blog-card');
    
    posts.forEach(post => {
        const postCategory = post.querySelector('.category').textContent;
        if (category === 'all' || postCategory === category) {
            post.style.display = 'block';
        } else {
            post.style.display = 'none';
        }
    });
};

// 검색 기능
const searchPosts = (query) => {
    const posts = document.querySelectorAll('.blog-card');
    
    posts.forEach(post => {
        const title = post.querySelector('h3').textContent.toLowerCase();
        const content = post.querySelector('p').textContent.toLowerCase();
        
        if (title.includes(query) || content.includes(query)) {
            post.style.display = 'block';
        } else {
            post.style.display = 'none';
        }
    });
};
